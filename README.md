# Tara Creatives — Landing Page

A single-file, Vercel-ready site for Tara Creatives.

## Quick Deploy

1. Create a new GitHub repo named `tara-creatives` (empty).
2. Upload `index.html` and `vercel.json` to the root.
3. In Vercel: **New Project → Import Git Repository → Framework: Other → Build command: (empty) → Output directory: /**.
4. Deploy. Your site will be live at `https://<project>.vercel.app`.

## Local Preview

Just open `index.html` in your browser.

## Edit Contact Details

In `index.html`, search and replace:
- `hello@taracreatives.com` with your real email
- `919000000000` with your WhatsApp number (no +)

## Optional

Add a custom domain in Vercel → Domains.
